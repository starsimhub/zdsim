

# ZDSim: Zero-Dose Simulation Model

## Overview
ZDSim is an **agent-based disease modeling framework** designed to identify and analyze **zero-dose children**—those who have never received any routine vaccinations. Using **tetanus vaccination data** as a proxy, the model simulates the impact of targeted immunization interventions to **reduce zero-dose prevalence among under-five children in Kenya**. 

This project leverages the **Starsim-SIS model** to estimate the number of **tetanus cases averted** through a **50% reduction in zero-dose vaccination rates** by 2025. The insights generated by ZDSim support public health decision-making by guiding immunization strategies to **increase vaccine coverage and optimize resource allocation**.

## Goals
- Develop a **disease model** to simulate **tetanus transmission and immunity** using agent-based modeling.
- Assess the **impact of vaccination interventions** on reducing zero-dose prevalence.
- Estimate the **number of tetanus cases prevented** by scaling up immunization efforts.
- Provide insights for **policy recommendations** to achieve WHO’s **Immunization Agenda 2030** targets.
- Expand the model to incorporate **additional vaccine-preventable diseases** such as diphtheria, pertussis, polio, and Haemophilus influenzae type b (Hib) in future iterations.

## Features
- **Agent-Based Modeling (ABM)** framework built using [Starsim](https://github.com/InstituteforDiseaseModeling/starsim).
- **Tetanus immunization data analysis** to infer zero-dose status.
- **Vaccination coverage estimation** based on tetanus as a reference indicator.
- **Intervention strategies** to improve routine immunization uptake.
- **Model calibration** using real-world epidemiological data.

## Installation
To set up the environment, clone this repository and install the required dependencies:

```bash
# Clone the repository
git clone https://github.com/starsimhub/zdsim.git
cd zdsim

# Create a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows use: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

## Usage
To run a basic simulation:

```bash
python run_simulation.py
```

The model outputs will provide insights into **zero-dose children and vaccination gaps**, helping shape **Pentacel vaccine interventions**.
